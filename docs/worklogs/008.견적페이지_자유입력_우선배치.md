# 008. 견적 페이지 자유입력 우선 배치 변경

## 작업일: 2026-02-11

## 변경 요약
견적 페이지(`/estimate`) 진입 시 자유입력 모드를 기본으로 표시하고, 입력창을 상단에 배치하여 즉시 입력 가능하도록 개선

## 변경 내용

### 1. 기본 입력 모드 변경
- **파일**: `src/stores/chatStore.ts`
- 초기 `inputMode`를 `'guided'` → `'free_text'`로 변경
- `clearChat()` 초기화 시에도 동일하게 `'free_text'`로 변경

### 2. 토글 순서 변경
- **파일**: `src/components/chat/ChatInputBar.tsx`
- [가이드 | 자유 입력] → [자유 입력 | 가이드] 순서로 변경

### 3. 입력창 상단 이동
- **파일**: `src/components/chat/ChatWindow.tsx`
- `ChatInputBar`를 하단(메시지 영역 아래)에서 상단(진행률 바 아래)으로 이동

### 4. 입력창 스타일 조정
- **파일**: `src/components/chat/ChatInputBar.tsx`
- `sticky bottom-0 border-t` → `sticky top-0 z-10 border-b`로 상단 고정 스타일 적용
- Textarea `min-h`를 `44px` → `72px`로 변경하여 약 2.5줄 높이로 확대

## 수정 파일
| 파일 | 변경 내용 |
|------|-----------|
| `src/stores/chatStore.ts` | 기본 inputMode를 free_text로 변경 |
| `src/components/chat/ChatInputBar.tsx` | 토글 순서 변경, 상단 고정 스타일, 입력창 높이 확대 |
| `src/components/chat/ChatWindow.tsx` | ChatInputBar 위치를 하단→상단으로 이동 |
